void sys_exit ( int  status)

Terminates the program.

This function terminates the program by printing the exit status, flushing the UART buffer, and disabling interrupts. It enters an infinite loop to halt execution.

Parameters
[in]statusThe exit status code.

Reads data from STDIN and echoes it back.

This function reads up to the specified number of bytes from the standard input (UART) into the buffer. It echoes the input back to the standard output.

Parameters
[in]fileThe file descriptor (must be 0 for STDIN).
[out]ptrPointer to the buffer where the input data will be stored.
[in]lenThe maximum number of bytes to read.
Returns
The number of bytes read on success, or -1 if the file descriptor is invalid.

◆ sys_sbrk()

void* sys_sbrk ( int  incr)

Adjusts the program break to allocate or deallocate memory.

This function shifts the program break to allocate or deallocate memory for the calling process. It ensures that the program break does not exceed the heap boundaries.

Parameters
[in]incrThe number of bytes to increase or decrease the program break.
Returns
A pointer to the previous program break on success, or (void*)-1 if the allocation fails.
