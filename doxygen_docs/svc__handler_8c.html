void svc_c_handler ( uint32_t *  stack_p)

Handles the SVC interrupt.

This function processes the SVC interrupt by extracting the SVC number and arguments from the stack frame and dispatching the appropriate system call.

Parameters
[in]stack_pPointer to the stack frame at the time of the SVC interrupt.
